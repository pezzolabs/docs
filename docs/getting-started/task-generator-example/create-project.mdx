# Step 1: Project Setup
:::note
This tutorial assumes you have already installed Pezzo. If you haven't, follow the [installation instructions](/docs/getting-started/installation/docker-compose).
:::

## 1. Sign in to Pezzo
Navigate to Pezzo at [https://localhost:4200](https://localhost:4200) and sign in (or sign up if you haven't already).


## 2. Create a new project
Create a new project and name it `Taskly`.
