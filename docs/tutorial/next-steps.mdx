# Next Steps

Taskly may be a simple demo app, but you can expand it further. Here are some ideas:

- Adjust your prompt to ask for the tasks to be created as user stories.
- Assign a difficulty level to each task.
- Ask OpenAI to provide a time estimate for how long each task will take.

The opportunities are endless with AI. Good luck!
