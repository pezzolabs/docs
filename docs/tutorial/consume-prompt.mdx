# Step 4: Consume Prompt

Now it's time to consume the prompt using the demo app.

## Install the demo app

All demo apps can be found in the [examples GitHub repository](https://github.com/pezzolabs/examples).

Open up a terminal and clone the app:

```bash
npx degit https://github.com/pezzolabs/examples/task-generator-app task-generator-app
```

Now, navigate to the app directory and install dependencies:

```bash
cd task-generator-app
npm install
```

## Environment variables
You'll need to provide the demo app with the following environment variables:
- `PEZZO_API_KEY`: Your Pezzo API key. You can copy this from the **Environments** menu in the sidebar.
- `OPENAI_API_KEY`: Your OpenAI API key.

```bash
export PEZZO_API_KEY=your-pezzo-api-key
export OPENAI_API_KEY=your-openai-api-key
```

## Run the demo app
You're ready to run the demo app!

```bash
npm run dev
```

The demo app should be available at [http://localhost:3100](http://localhost:3100). Try it out!
